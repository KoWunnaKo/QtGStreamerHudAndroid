HUD Setup Procedure:
To enable video, you must supply a valid GStreamer pipeline string. To change the string, select the Settings menu in the upper right corner of the display. Please refer to the GStreamer documentation at http://gstreamer.freedesktop.org for more information on GStreamer pipelines.
The following is an example pipeline string:

udpsrc port=9000 buffer-size=60000 ! application/x-rtp,encoding-name=H264,payload=96 ! rtph264depay ! h264parse ! queue ! avdec_h264

The following will display a test pattern:

videotestsrc ! queue

Note: The final element in a normal GStreamer pipeline is the 'Video Sink' element. Do NOT supply a video sink, as the application will supply this automatically. The example string is for a UDP connection to an external source using H264 encoding.

After changing the pipeline string, you must stop any running streams, and re-start using the toolbar in the settings menu.

MAV-Link Setup Procedure:

To enable the MAV-Link stream, you must connect to a UAV source by one of two methods: UDP socket stream or TCP socket stream. In the settings menu, enter a connect string in the 'MAV Connect' field using the following format:

For a UDP connection: UDP:{port}     For example UDP:14550

For a TCP connection: TCP:{host/IP address}:{port}    For example TCP:192.168.0.1:9000

Note: The HUD will automatically listen on UDP port 14550, the standard default UDP port for a MAV-Link connection, therefore it is not necessary to configure a UDP connection unless you want to use a different port than the default.

 Toolbar Description:

    Play video (start GStreamer pipeline)
    Stop video (stop GStreamer pipeline)
    Pause video (pause GStreamer pipeline)
    Reset sliders to default position
    Connect to MAV-Link stream
    Disconnect from MAV-Link stream
